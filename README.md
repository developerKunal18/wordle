# wordle
your own clone of the viral WORDLE game, entirely in Python, playable in the terminal! No libraries. No GUI. Just pure logic &amp; fun.  ✔ 5-letter word guessing game ✔ Green/Yellow/Gray style hints ✔ Random word selection ✔ Limited attempts (6 like real Wordle) ✔ Dictionary-based validation ✔ Perfect for GitHub + portfolio
